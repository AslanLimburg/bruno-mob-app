📋 3 СПОСОБА ПОЛУЧИТЬ APK:
═══════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────┐
│ 🌟 СПОСОБ 1: GITHUB ACTIONS (Автоматически, онлайн)   │
│    ⏱️  Время: 10-15 минут                               │
│    💰 Стоимость: БЕСПЛАТНО                              │
└─────────────────────────────────────────────────────────┘

1. Создайте GitHub репозиторий (если ещё нет)
2. Загрузите проект:
   
   cd /Users/user/Desktop/bt
   git init
   git add .
   git commit -m "Android app ready"
   git remote add origin https://github.com/ваш-username/bruno-token.git
   git push -u origin main

3. Перейдите на GitHub → Actions
4. Запустите workflow "Build Android APK"
5. Скачайте готовый APK из Artifacts

┌─────────────────────────────────────────────────────────┐
│ 💻 СПОСОБ 2: ЛОКАЛЬНО (Android Studio)                 │
│    ⏱️  Время: 1 час (установка) + 10 мин (сборка)       │
│    💰 Стоимость: БЕСПЛАТНО                              │
└─────────────────────────────────────────────────────────┘

1. Установите Android Studio:
   https://developer.android.com/studio

2. Откройте проект:
   
   cd /Users/user/Desktop/bt/frontend
   npx cap open android

3. В Android Studio:
   Build → Build Bundle(s) / APK(s) → Build APK(s)

4. APK в: android/app/build/outputs/apk/debug/app-debug.apk

┌─────────────────────────────────────────────────────────┐
│ ⚡ СПОСОБ 3: КОМАНДНАЯ СТРОКА (Gradle)                 │
│    ⏱️  Время: 30 мин (установка Java) + 5 мин (сборка)  │
│    💰 Стоимость: БЕСПЛАТНО                              │
└─────────────────────────────────────────────────────────┘

1. Установите Java:
   
   brew install openjdk@17
   echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 17)' >> ~/.zshrc
   source ~/.zshrc

2. Соберите APK:
   
   cd /Users/user/Desktop/bt/frontend/android
   ./gradlew assembleDebug

3. APK в: app/build/outputs/apk/debug/app-debug.apk

🚀 ПУБЛИКАЦИЯ В МАГАЗИНАХ:

▶ GOOGLE PLAY:

1. Зарегистрируйтесь: https://play.google.com/console ($25)
2. Соберите подписанный AAB:
   ./gradlew bundleRelease
3. Загрузите AAB в консоль
4. Заполните информацию + Privacy Policy
5. Отправьте на проверку (1-7 дней)

▶ APPGALLERY (HUAWEI):

1. Зарегистрируйтесь: https://developer.huawei.com (БЕСПЛАТНО)
2. Используйте тот же APK что и для Google Play
3. Загрузите в AppGallery Connect
4. Заполните информацию
5. Отправьте на проверку (1-3 дня)
