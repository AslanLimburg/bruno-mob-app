🎯 ПОШАГОВАЯ ИНСТРУКЦИЯ:

ШАГ 1: Откройте BrunoChat
───────────────────────────
   ➜ http://localhost:3000
   ➜ Войдите в систему
   ➜ Перейдите в Messenger (💬)


ШАГ 2: Начните новый чат
───────────────────────────
   
   СПОСОБ A - Кнопка на экране:
   ┌─────────────────────────────────┐
   │         💬                      │
   │      BrunoChat                  │
   │                                 │
   │  [➕ Start New Chat]  ← НАЖМИ! │
   └─────────────────────────────────┘

   СПОСОБ B - Кнопка в шапке:
   ┌─────────────────────────────────┐
   │ 💬 BrunoChat      [➕]  ← НАЖМИ!│
   └─────────────────────────────────┘


ШАГ 3: Найдите пользователя
───────────────────────────
   ┌─────────────────────────────────┐
   │ [🔍 Search users...]            │ ← Введите минимум 2 символа
   │                                 │
   │ [A] Alice (alice@test.com)      │ ← Кликните на пользователя
   │ [B] Bob (bob@test.com)          │
   └─────────────────────────────────┘

   Попробуйте:
   - "test"
   - "demo"
   - "admin"
   - Или любое имя/email


ШАГ 4: Кликните на найденного пользователя
───────────────────────────────────────────
   ✅ Откроется окно чата
   ✅ Поле ввода внизу станет БЕЛЫМ
   ✅ Можно печатать!


ШАГ 5: Введите сообщение
───────────────────────────
   ┌─────────────────────────────────┐
   │ Alice Smith     👤  🔥 20min    │ ← Шапка чата
   ├─────────────────────────────────┤
   │                                 │
   │ (Здесь будут сообщения)         │
   │                                 │
   ├═════════════════════════════════┤ ← СИНЯЯ ГРАНИЦА
   │ 😀 [Type a message...] 📎 📤   │ ← ПОЛЕ ВВОДА (БЕЛОЕ!)
   └─────────────────────────────────┘

   Кликните в поле, введите текст
   Нажмите Enter или кнопку 📤

🔴 ПОЧЕМУ ПОЛЕ НЕАКТИВНО (СЕРОЕ)?

   ❌ Контакт НЕ выбран

   Решение:
   1. Нажмите ➕ в шапке (или кнопку "Start New Chat")
   2. Введите имя пользователя (минимум 2 символа)
   3. КЛИКНИТЕ на найденного пользователя
   4. Поле станет БЕЛЫМ ✅

🚀 БЫСТРЫЙ СТАРТ

1. Обновите страницу: Cmd+Shift+R (Mac) или Ctrl+Shift+F5 (Win)

2. Вы увидите:
   ✅ Желтую подсказку над полем ввода
   ✅ Кнопку "➕ Start New Chat" в центре
   ✅ Синюю границу над полем ввода

3. Нажмите "➕ Start New Chat" или ➕ в шапке

4. Введите "test" или "demo" в поиск

5. Кликните на найденного пользователя

6. Поле станет БЕЛЫМ - можно печатать! ✅
