1️⃣  ПОЛУЧИТЬ КЛЮЧ (5 минут)
   
   → Откройте: https://platform.openai.com/api-keys
   → Войдите или зарегистрируйтесь  
   → Нажмите "Create new secret key"
   → Скопируйте ключ (sk-proj-...)

2️⃣  ДОБАВИТЬ В ПРОЕКТ (1 минута)

   → Откройте: backend/.env
   → Найдите строку: OPENAI_API_KEY=...
   → Вставьте ваш новый ключ
   → Сохраните файл

3️⃣  ПЕРЕЗАПУСТИТЬ (30 секунд)

   → Терминал:
   pkill -f "node server.js"
   cd /Users/user/Desktop/bt/backend
   npm start
