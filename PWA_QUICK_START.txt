🎯 ЧТО ТАКОЕ PWA?

   Progressive Web App - это веб-приложение которое можно:
   
   📱 Установить на устройство (Desktop, Mobile)
   ⚡ Использовать офлайн (без интернета)
   🔔 Получать push-уведомления
   🚀 Запускать с домашнего экрана

Frontend:

   ✅ public/service-worker.js - Service Worker
   ✅ src/serviceWorkerRegistration.js - регистрация SW

🚀 КАК УСТАНОВИТЬ

Desktop (Chrome, Edge):
──────────────────────
   1. Откройте http://localhost:3000
   2. В адресной строке появится иконка ➕
   3. ИЛИ в правом нижнем углу кнопка:
      ┌─────────────────────────┐
      │ 📱 Install Bruno Token  │
      │    Add to Home Screen   │
      └─────────────────────────┘
   4. Нажмите "Install"
   5. Приложение откроется в отдельном окне
   6. Иконка появится на рабочем столе/в меню

Mobile Android (Chrome):
────────────────────────
   1. Откройте в Chrome
   2. Появится баннер снизу "Install Bruno Token"
   3. ИЛИ кнопка в правом нижнем углу
   4. Нажмите "Install"
   5. Иконка появится на главном экране

Mobile iOS (Safari):
─────────────────────
   1. Откройте в Safari
   2. Нажмите кнопку "Share" (поделиться) внизу
   3. Scroll down → "Add to Home Screen"
   4. Введите название (Bruno Token)
   5. Нажмите "Add"
   6. Иконка появится на главном экране

🎨 ФУНКЦИИ PWA

✅ Установка на устройство
   - Desktop: Windows, Mac, Linux
   - Mobile: Android, iOS (16.4+)

✅ Офлайн режим
   - Статические страницы работают без интернета
   - Кэш обновляется автоматически

✅ App Shortcuts (быстрые действия)
   - Dashboard
   - Messenger
   - Lottery
   - Challenge

✅ Standalone режим
   - Нет браузерного UI
   - Полноэкранное приложение
   - Как нативное приложение

✅ Auto-updates
   - Обновления скачиваются автоматически
   - Уведомление о новой версии

⏳ Push Notifications (требует backend setup)
   - Новые сообщения
   - Результаты лотереи
   - Обновления Challenge

🎯 БЫСТРЫЙ ТЕСТ

1. Обновите страницу (Hard Refresh):
   Mac: Cmd + Shift + R
   Win: Ctrl + Shift + F5

2. Проверьте консоль (F12 → Console):
   Должно быть:
   ✅ "🚀 PWA Service Worker installing..."
   ✅ "📦 Opened cache"
   ✅ "✅ PWA Service Worker activated"
   ✅ "🎉 Bruno Token PWA Service Worker loaded successfully!"

3. Проверьте кнопку установки:
   Правый нижний угол → "📱 Install Bruno Token"

4. Нажмите и установите!
