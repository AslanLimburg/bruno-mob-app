💡 КАК ИСПОЛЬЗОВАТЬ

1. ОТПРАВИТЬ СООБЩЕНИЕ
   - Выберите контакт из списка
   - Введите текст
   - Нажмите Enter или кнопку 📤

2. ДОБАВИТЬ EMOJI
   - Нажмите кнопку 😀
   - Выберите категорию (вкладки)
   - Кликните на emoji

3. ЗАГРУЗИТЬ ФАЙЛ
   - Нажмите кнопку 📎
   - Выберите файл (до 10 MB)
   - Preview появится
   - Нажмите "📤 Send File"

4. ПРОСМОТРЕТЬ ПРОФИЛЬ
   - Кликните на имя контакта в шапке чата
   - Sidebar откроется справа
   - Информация + действия

5. ДОБАВИТЬ КОНТАКТ
   - Нажмите ➕ в шапке
   - Введите имя или email в поиск
   - Нажмите ➕ рядом с нужным пользователем

🎯 ТЕХНИЧЕСКИЕ ДЕТАЛИ

Загрузка файлов:
- Multer storage: /uploads/messenger/
- Filename: msg-{timestamp}-{random}.{ext}
- Max size: 10 MB
- Allowed: jpeg, jpg, png, gif, webp, mp4, mov, avi, mp3, wav, 
           pdf, doc, docx, txt

Emoji:
- Total: 200+
- Categories: 7
- Grid: 8 columns (desktop)
- Responsive: 5-8 columns

Анимации:
- slideIn, fadeIn, bounce, pulse
- float, slideUp, slideInRight
- Smooth transitions на всех элементах
